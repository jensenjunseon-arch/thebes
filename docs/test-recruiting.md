# 테스터 모집 키트 (1단계 · 수요 검증)

목표: **10~20명**에게 무료 진단을 써보게 하고, "사고 싶어 하는지"를 확인한다.
진단 링크: **https://thebes-nine.vercel.app/session/demo**

> 핵심 원칙: 아직 더 만들지 마세요. 이 링크 하나로 검증이 됩니다.
> 말로 하는 "좋아요"보다, **끝까지 했는지 / 결제 의향을 눌렀는지**가 더 진짜 신호예요.

> 솔직한 피드백이 너무 필요해서요 🙏 5분만 써보고 끝에 한마디만 남겨주실 수 있을까요?

> 써보시고 끝에 30초 설문(별로/좋아요, 한마디)만 남겨주시면 큰 힘이 됩니다.

## 4. 다 해본 뒤 직접 물어볼 4가지 (통화·대면)

진단 끝의 30초 설문이 자동 저장되지만, 1:1로 더 물어보면 가장 값집니다.

1. **끝까지 했어요?** 안 했다면 — 어디서, 왜 멈췄나요?
2. **나(아이)에 대해 새로 안 게 있었나요?** ‘리포트’가 더 보고 싶었나요?
3. 이거 **주변에 추천**할 것 같아요? (1~10점)
4. **월 19,900원이면 결제**할 것 같아요? 아니라면 얼마면 할까요?

---

## 5. 결과 읽는 법 (신호 해석)

- **완주율**: 링크 보낸 사람 중 끝까지 한 비율. 50% 넘으면 훅이 통한 거예요.
- **결제 의향 ‘네’ 비율**: 30~40% 넘으면 강한 신호. (단, 말뿐인 의향은 부풀려져요 — 진짜는 "어떻게 결제해요?"를 먼저 묻는 사람)
- **자유 한마디**: 같은 불만/같은 칭찬이 3번 이상 반복되면, 그게 다음 작업 1순위.
- 응답은 **Supabase → Table Editor → `feedback`** 에서 봅니다. (마이그레이션 `20260603100000_feedback.sql` 실행 필요)
